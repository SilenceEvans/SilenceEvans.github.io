二、解决连接mysql数据库时遇到的Can’t connect to local MySQL server through socket ‘/tmp/mysql.sock’

在重新安装mysql后，我在使用:

登陆数据库时碰到了该问题，其实我这个问题的产生就是因为我数据库并未完全安装成功导致的。

所以你如碰到了该问题，请注意一下在你重新安装数据库时是否碰到了我上面第一部分提到的未完全安装成功的问题。

一定要使用mysql.server start命令启动数据库，看是否报错，如果使用brew services start mysql后台启动数据库时，是无法看到启动失败的问题的，这时连接数据库也会出现Can’t connect to local MySQL server through socket ‘/tmp/mysql.sock’ 该问题。
